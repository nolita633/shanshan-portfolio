# 刘姗姗个人展示网站开发计划（修订版）

## 设计理念

* **克制美学**：奶白色 (#F5F2ED) 背景 + 深木炭色 (#1A1A1A) 文字，高级纸质媒体质感

* **中英结合**：大章节标题使用"中文 + 英文副标题"

* **呼吸感**：大量留白，大间距

* **字体**：

  * 中文标题：Noto Serif SC

  * 英文标题：Playfair Display

  * 正文：Inter

## 技术栈

* Next.js 14 (App Router)

* TypeScript

* Tailwind CSS

* Framer Motion

## 网站结构

### 1. 首页/英雄区域

* 大排版姓名展示

* 职位副标题动画

* 优雅的滚动指示器

### 2. 关于我 ABOUT

* 卡片式布局，悬停微交互

* 技能标签平滑动画

* 个人照片（画报风格）

### 3. 工作经历 WORK EXPERIENCE

* 优雅的时间线布局

* 大厂Logo精致展示

* 悬停展开详情

* 数据成果可视化（克制设计）

### 4. 项目案例 PROJECTS

* 画报风格卡片网格

* 悬停时的微妙动画

* 富媒体展示（图片/GIF）

### 5. AI助手 AI Assistant

* 右下角优雅的聊天窗口

* 基于简历的智能回复

* 流畅的打字机动画

### 6. 联系 CONTACT

* 极简社交链接

* 优雅的表单设计

## 开发步骤

1. 初始化Next.js 14项目（App Router）
2. 配置Tailwind CSS和字体
3. 创建基础布局组件
4. 实现各页面区块
5. 添加Framer Motion动画
6. 实现AI助手功能
7. 优化和测试
